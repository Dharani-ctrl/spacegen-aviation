================================================================================
                         QUICK START ✅
================================================================================

FRONTEND SETUP:
  1. npm install
  2. npm run dev
  3. Visit http://localhost:3000

BACKEND SETUP:
  1. cd server
  2. npm install
  3. cp .env.example .env
  4. Edit .env (add MONGODB_URI, JWT_SECRET)
  5. npm run dev
  6. Backend runs on http://localhost:5000

MONGODB SETUP:
  1. Go to https://mongodb.com/cloud/atlas
  2. Create free M0 cluster
  3. Get connection string
  4. Add to backend .env as MONGODB_URI

TESTING:
  1. Submit enquiry form on website
  2. Check backend for data
  3. Login to admin panel
  4. View & manage enquiries

================================================================================
                      DEPLOYMENT STEPS ✅
================================================================================

FRONTEND → VERCEL (5 minutes):
  1. Push to GitHub
  2. Connect repo to Vercel
  3. Set NEXT_PUBLIC_API_URL env var
  4. Click Deploy!

BACKEND → RENDER (10 minutes):
  1. Push /server folder to GitHub
  2. Connect repo to Render
  3. Set env variables (MONGODB_URI, JWT_SECRET, etc.)
  4. Click Create Web Service!

DATABASE → MONGODB ATLAS (5 minutes):
  1. Create account & cluster
  2. Get connection string
  3. Add to backend env vars
  4. Done!

TOTAL SETUP TIME: ~20 minutes for production!
